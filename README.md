# BuyHome
本專案實作一個小而完整的購物車 APP。
本 APP 於測試環境下進行 google pay 付款，故不會真正扣款，敬請放心。

# 使用技術
- MVVM / ViewModel / Fragment / Navigation
- View Binding / Data Binding / LiveDate
- RecyclerView / ListView / Custom Adapter
- ViewPager / CardView
- Firebase / Google Sign-In API
- Google Pay API
- SharedPreferences
